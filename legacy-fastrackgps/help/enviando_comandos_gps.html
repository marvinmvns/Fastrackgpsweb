Sobre os comandos
Rastrear uma vez: Mostra um único ponto no mapa, indicando o local onde se encontra o bem.

Rastrear a cada N (Segundo/Minuto/Hora): Indica de quanto em quanto tempo será mostrado um novo ponto no mapa.

Velocidade limite: Programa o gps para enviar alertas se a velocidade requerida for ultrapassada.

Desabilitar: Desabilita o rastreamento.

Alerta movimento: Programa o gps para enviar alertas se o bem entrar em movimento. Use este comando quando o bem se encontrar parado.

Cancelar alerta: Cancela todos os alertas (Velocidade e Movimento).

Modo SMS: Programa o gps para operar em modo SMS, ou seja, cancela o GPRS. Neste modo não é possível fazer o rastreamento. Para reativar o modo GPRS, envie para o aparelho o comando via sms: adminip123456 XXX.XXX.XXX.XXX PPPP, onde XXX equivale ao IP do servidor, e PPPP à porta escolhida.
